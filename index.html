Conexión ESP32 a Wi-Fi

Aprende a conectar tu ESP32 a una red Wi-Fi y empieza a crear proyectos IoT innovadores

¿Para qué sirve conectar el ESP32 a Wi-Fi?

El **ESP32** es un microcontrolador versátil que ofrece conectividad **Wi-Fi** y **Bluetooth**, ideal para aplicaciones IoT. Al conectar el ESP32 a una red Wi-Fi, se pueden realizar proyectos como:

  • Automatización del hogar: Controla dispositivos electrónicos desde cualquier lugar.
  • Monitoreo remoto: Accede a sensores y datos desde cualquier parte del mundo.
  • Integración con la nube: Envía datos a plataformas como ThingSpeak o Blynk.
  • Comunicación entre dispositivos: Conecta varios dispositivos ESP32 para crear redes IoT.

Código de Conexión Paso a Paso

A continuación, te mostramos el código básico para conectar el ESP32 a una red Wi-Fi:

#include 

const char* ssid = "Tu_Nombre_de_Red";  // Nombre de tu red Wi-Fi
const char* password = "Tu_Contraseña";  // Contraseña de tu red Wi-Fi

void setup() {
  Serial.begin(115200);  // Inicializa la comunicación serial
  delay(10);

  // Conecta a la red Wi-Fi
  Serial.println("Conectando a Wi-Fi...");
  WiFi.begin(ssid, password);

  // Espera hasta que esté conectado
  while (WiFi.status() != WL_CONNECTED) {
    delay(1000);
    Serial.println("Intentando conectar...");
  }

  // Una vez conectado, imprime la dirección IP
  Serial.println("Conexión exitosa!");
  Serial.print("Dirección IP: ");
  Serial.println(WiFi.localIP());
}

void loop() {
  // El código de tu aplicación va aquí
}

Este código es sencillo de usar. Solo debes sustituir "Tu_Nombre_de_Red" y "Tu_Contraseña" con los datos de tu red Wi-Fi. Luego, sube el código a tu ESP32 y abre el Monitor Serie para obtener la dirección IP asignada.

Puente Serial a Bluetooth Clásico (SPP)

Este código establece un puente entre el puerto serial del ESP32 y el Bluetooth clásico usando el perfil SPP:

//This example code is in the Public Domain (or CC0 licensed, at your option.)
//By Evandro Copercini - 2018

#include "BluetoothSerial.h"

#if !defined(CONFIG_BT_ENABLED) || !defined(CONFIG_BLUEDROID_ENABLED)
#error Bluetooth is not enabled! Please run `make menuconfig` to and enable it
#endif

BluetoothSerial SerialBT;

void setup() {
  Serial.begin(115200);
  SerialBT.begin("ESP32test"); //Bluetooth device name
  Serial.println("The device started, now you can pair it with bluetooth!");
}

void loop() {
  if (Serial.available()) {
    SerialBT.write(Serial.read());
  }
  if (SerialBT.available()) {
    Serial.write(SerialBT.read());
  }
  delay(20);
}

Este código permite crear una comunicación bidireccional entre el puerto serial del ESP32 y un dispositivo Bluetooth clásico, como un teléfono o una computadora.
